# **Ecommerce Blinkit Sales Dashboard**

## **Overview**

This project is a Power BI dashboard created to analyze and visualize Blinkit sales data.  
The dashboard provides insights into sales performance, product categories, customer behavior, and business trends using interactive visualizations.

---

## **Features**

- Sales performance analysis  
- Product category insights  
- Revenue and profit tracking  
- Interactive filters and slicers  
- Customer and order analysis  
- Data visualization using Power BI  

---

## **Tools & Technologies**

- Power BI  
- Data Visualization  
- DAX  
- Excel / CSV Dataset  
- Business Intelligence  

---

## **Dashboard Insights**

The dashboard helps in:

- Identifying top-selling products  
- Understanding customer purchasing patterns  
- Tracking total sales and revenue trends  
- Comparing category-wise performance  
- Supporting data-driven business decisions  

---

## **Project File**

`Ecommerce blinkit dashboard.pbix`

---

## **How to Use**

1. Download the `.pbix` file from this repository  
2. Open it using Power BI Desktop  
3. Explore the interactive dashboard visuals
